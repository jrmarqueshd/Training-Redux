# ReduxPlace

Link da aplicação: https://reduxplace.ga/

Este projeto foi desenvolvido para colocar em prática os meus estudos em Redux! <br><br> A ideia a princípio foi criar uma vitrine de produtos com dados 'mocados' e simular a ação de adicionar os produtos ao carrinho... Até o momento todo o estado dessa aplicação está sendo controlado via Redux, sendo assim, nessa aplicação só tem Stateless Components (_Componentes sem estados_), uma vez que todos os estados necessários para que a aplicação funcione está no Redux.

### Tecnologias utilizadas:

- Redux;
- React Redux;
- Styled Components;
- React Icons.
